Courses

Data Structures

This course emphasizes the study of the basic data structures of computer science (stacks, queues, trees, lists, graphs) and their implementations using the Java language. Included in this study are programming techniques which use recursion and reference variables. Students in this course are also introduced to various searching and sorting methods and are also expected to develop an intuitive understanding of the complexity of these algorithms.

Algorithms Implementation

This course covers a broad range of the most commonly used algorithms.The students will implement and test several algorithms. The course is programming intensive. The purpose of the course is to teach the basics of how to convert algorithms into programs. The course reveals that simple algorithms can become complicated when attempting to implement them, complex algorithms must be considered carefully during implementation, and issues during implementation are always bound to occur. The course also teaches the differences in algorithms and how they affect run-times of the associated programs. Furthermore, that different algorithms can be used to solve the same problem and different solutions can be compared by various metrics. Major topics for the course includes the following: sorting, searching, hashing, compression, heaps and priority queues, graph algorithms, dynamic programming, P vs NP, heuristic approximation, and cryptography.

Computer Organization and Assembly Language

The purpose of this course is to study the components of computing systems common to most computer architectures. In particular, this class is meant to introduce data representation, types of processors (e.g., RISC V. CISC), memory types and hierarchy, assembly language, linking and loading, and an introduction to device drivers.




























Projects

Pitt-Yak

This was a group project for the Programming for Web Applications course at the University of Pittsburgh. The goal of the project was to create and build a website with your group mates by the end of the term. My group which consisted of myself, Joesph McClain, Philip Washy, and Matthew O'Brian came up with the idea of creating a localized version of the well known iPhone and Andriod app known as "Yik-Yak". Similar to Yik-Yak our website is an anonymous social media website that allows users to up-vote and down-vote posts and subposts. Furthermore, these posts are tracked by locations around the Pittsburgh campus, such as North Oakland, South Oakland, and Central Oakland. This gives users a a general idea of what is going on at each area around campus. We used the Python flavor of Google App Engine for our back-end, in combination with HTML5, CSS, JavaScript, BootStrap, & JQuery for the front-end.


Magic Mirror

This was an individual project (still in progress) that I researched and developed solely for myself. While wanting to get my hands on the latest technology, I stumbled upon a device called the Raspberry Pi. The Raspberry Pi is a single board mini computer that is capable of performing a bunch of different tasks. As a result, I looked into various projects for the Raspberry Pi that I would like to work on and I found this one in particular. While I havent gotten my hands on a Raspberry Pi just yet, I have began to work on this project and have almost gotten the interface for the mirror completed. Mine interface is a little bit different then that pictured. Mine will be displaying the Time and Date, Weather for the week, Top News Headlines of the day, and my own Nike FuelBand Statistics for the week. This is still a work in progress but hopefully it will be complete soon! You can read more about the project by checking out the link to Raspberry Pi's blog!
